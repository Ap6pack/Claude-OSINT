# Privacy Policy

**outrider-recon** is an open-source collection of OSINT skill files for Claude Code. It does not collect, store, transmit, or process any user data.

## What this plugin does

- Provides plain Markdown skill files that Claude loads as context
- Includes an optional MCP server that makes outbound API requests to public OSINT services (crt.sh, HudsonRock, FIRST EPSS, Wayback Machine, DNS resolvers) on your behalf when you invoke those tools

## What this plugin does not do

- No telemetry or analytics
- No user accounts or authentication
- No data collection or tracking
- No cookies or local storage
- No network requests unless you explicitly invoke an MCP tool

## MCP server requests

When you use the optional MCP server, requests are made directly from your machine to public APIs. No data passes through any intermediary or third-party service operated by the plugin author.

## Contact

For questions, open an issue at https://github.com/Ap6pack/outrider-recon/issues.
